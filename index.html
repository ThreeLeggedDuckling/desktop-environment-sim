Mon Bureau

A propos

Mon Bureau est une simulation permettant aux personnes de se familiariser avec l'interface d'un bureau d'ordinateur et l'utilisation de la sourisclic gauche, clic droit, double clic, glisser déposer.

Les apprenants peuvent explorer librement les manipulations dans le mode bac à sable ou s'entraîner sur des actions spécifiques dans le mode exercices.

Contrôles

  • Passer le jeu en plein écran : touche ESPACE
  • Quitter le plein écran : touche ECHAP
  • Sélectionner un mode de jeu : CLIC GAUCHE ou touche du pavé numérique correspondante
